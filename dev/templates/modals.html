Що таке Frontend?

Frontend — це частина веброзробки, яка відповідає за все, що бачить і з чим взаємодіє користувач у браузері. Це візуальна частина сайту або застосунку, включно з розміткою, стилями, анімаціями та логікою інтерфейсу.

  • HTML — створює каркас сторінки (текст, кнопки, блоки)
  • CSS — додає кольори, шрифти, відступи, робить сайт красивим
  • JavaScript — дозволяє сайту реагувати на дії (клік, прокрутка, анімації)
  • Git — зберігає історію змін у коді, щоб нічого не загубилось
  • Chrome DevTools — набір інструментів для перевірки і налагодження сайту
